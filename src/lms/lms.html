What is the LMS used for?
It is the application that WTC provides for distributing projects, tracking progress, saving, grading and reviewing your work. Without it you will not be able to start or submit any work. Correct use of the LMS will be needed to work through the curriculum without any issues.
Basics you will need to know before continuing.
tilde
- symbol stands for your home directory. If you are user, then the tilde (~) stands for /home/user
pwd
- Lets you to know in which directory you're located (pwd stands for "print working directory"). Example: "pwd" in the Desktop directory will show "~/Desktop". Note that the GNOME Terminal also displays this information in the title bar of its window. A useful mnemonic is "present working directory
ls
- The command will show you ('list') the files in your current directory. Used with certain options, you can see sizes of files, when files were made, and permissions of files. Example: ls ~ will show you the files that are in your home directory.
cd
- The command will allow you to change directories. When you open a terminal you will be in your home directory. To move around the file system you will use cd. Examples:
cd / To navigate into the root directory
cd ~ or cdTo navigate to your home directory
cd .. To navigate up one directory level
cd - To navigate to the previous directory
