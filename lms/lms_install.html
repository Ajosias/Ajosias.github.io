Installing the LMS

What is the LMS used for?
It is the application that WTC provides for distributing projects, tracking progress, saving, grading and reviewing your work. Without it you will not be able to start or submit any work. Correct use of the LMS will be needed to work through the curriculum without any issues.
Installing the LMS
Please note: You need have downloaded the newest version of the LMS file (to be found on Slack). You also need to make sure you have a bin file in your home directory, if not please return to the lms basics page.
